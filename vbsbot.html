The VBSBot 
    It isn't a real bot, but it can be used for automating stuff 
    Replace "H" after WshShell.SendKeys method with anything you want to type

Set WshShell = WScript.CreateObject("WScript.Shell")
WshShell.Run "notepad"
WScript.Sleep 100
WshShell.AppActivate "Notepad"
WScript.Sleep 500
WshShell.SendKeys "H"
WScript.Sleep 500


This types h
